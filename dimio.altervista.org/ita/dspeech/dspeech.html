* INSTALL VOICES IN WINDOWS. *

Voices you have downloaded have to been installed in Windows as standard programs. Now, on drop down menu of DSpeech, you can see all new installed voices. If is not so, try to restart the program.

* START DSPEECH *

From this link, you can download the last version of DSpeech, this program is stand alone (portable), so you do not have to instal it.

You have just to unzip the Zip where you want (Desktop) and double click on DSpeech file.

Now:

  • Select from drop down menu the voice you want to use (right picture).
  • Write something on main DSpeech window or open the file you want read.
  • Press "Speak from start" button as shown in the picture below.

     NOW DSPEECH WILL SPEAK IN SELECTED LANGUAGE

*  HOW TO BUILD DIALOGUES  *

By the context menu you can change voice and create dialoques between different voices and languages.

To insert a change voice Tag, you have to right click on text window and select on voice you want to insert.

On selected position, will be created a new voice change TAG as this:

>VOICE Juliette

DSpeech will read this line as a "change voice command" and will switch to use selected voice.
It's important that you don't put any other word on this line, because TAG commands want to be alone on line.



* HOW TO TRANSLATE INTERFACE TO OTHER LANGUAGE *

If you want you can translate the DSpeech's interface to another language, so for example if you select Italian, the button "Show Character" will be converted to "Mostra Personaggio".
